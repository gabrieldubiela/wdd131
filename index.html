Welcome to WDD 131 - Dynamic Web Fundamentals

This course is designed to provide students with the knowledge and skills necessary to create dynamic web pages using HTML, CSS, and JavaScript. Students will learn how to create interactive web pages that respond to user input and provide a rich user experience. Topics covered include HTML5, CSS3, JavaScript, jQuery, and AJAX.
