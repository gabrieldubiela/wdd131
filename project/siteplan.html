Color Schema

--primary-color: #12103d; (Dark Navy/Almost Black):

header: Background color.

h2: Text color.

header button: Background color.

main button: Background color.

footer: Background color.

--secondary-color: #cbba64; (Golden Tan/Beige):

h1: Text color.

header button: Border color (left side).

main button: Text color.

footer: Text color.

--background-color: #f9f9f9; (Light Gray/Off-White):

body: Background color.

header button: Text color.

section: Background color.

--text-color: #333; (Dark Gray):

body: Text color (all paragraph text, unless overridden).

Typograph

--title-font: "Roboto", sans-serif;: Roboto is used for all titles and the header button.

h1: Main title on the page (2rem size).

h2: Section titles (1.5rem size).

header button: The button in the header navigation (1.4rem size).

--text-font: "DM Serif Text", serif;: DM Serif Text is used for all the main body text.

body: The overall body text (default size, likely 1rem).

p: Paragraph text (1rem size).

footer p: Footer paragraph text (0.9rem size).
